# Batch Translation

Batch translation lets Codex queue AI suggestions for several cells instead of working one cell at a time. It is useful for first drafts, repeated structures, and sections where the project system message is already strong.

## Before You Start

1. Confirm source and target languages are correct in the Main Menu.
2. Review [AI Settings](/docs/translation/ai-settings).
3. Make sure the current file has enough context and examples.
4. Decide which cells should be included.

## Run Batch Translation

1. Open the file you want to translate.
2. Use **Autocomplete Chapter** from the chapter or section toolbar.
3. Choose which cells to include, such as empty cells or cells that have not been validated.
4. Choose how many cells to queue.
5. Start the batch.
6. Review each suggestion before validating it.

<Callout type="warning">
Do not run a large batch immediately after changing the system message. Test a few cells first so you can catch tone, terminology, or formatting problems early.
</Callout>

## Per-Cell Autocomplete vs Chapter Autocomplete

- **Autocomplete with AI** works on one selected cell.
- **Autocomplete Chapter** queues multiple cells in the current chapter or section.

Use per-cell autocomplete for sensitive passages or cleanup. Use chapter autocomplete when the pattern is predictable and review capacity is available.

## Review Checklist

After a batch finishes:

- Read each generated cell against the source.
- Fix terminology and formatting.
- Check footnotes, tags, or HTML if the source uses them.
- Add comments for questions.
- Validate only reviewed work.

## Troubleshooting

If batch translation is unavailable, confirm a target file is open, you are signed in, and the project has a system message. If suggestions are poor, update [AI Settings](/docs/translation/ai-settings) before trying again.
