## Before You Escalate

Use the guided flow above, then make sure you have:

1. Checked for updates with the [update guide](/docs/project-management/update-extensions)
2. Compared your issue against the [latest release notes](/docs/releases/latest)
3. Copied the support handoff template with version, OS, reproduction steps, and results
4. Reviewed [Reporting Bugs](/docs/project-management/reporting-bugs) if you need help collecting logs or evidence

<Callout type="warning">
The fastest support requests include exact version details, a clear reproduction path, and the full error text when available.
</Callout>

If you still need help, start with [Discord](https://discord.gg/6kVJTEXYEp) for triage. When the problem is confirmed as a bug, use [Reporting Bugs](/docs/project-management/reporting-bugs) to decide whether it belongs in [Codex Editor issues](https://github.com/genesis-ai-dev/codex-editor/issues) or [Codex application issues](https://github.com/genesis-ai-dev/codex/issues).
