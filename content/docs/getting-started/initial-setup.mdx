# Initial Setup

## Download Codex Editor

Before getting started, you'll need to [download Codex Editor](https://codexeditor.app) if you haven't already.

3. **Restart if Needed**: If the application doesn't load properly, close and reopen it

## System Requirements

Before installation, ensure your system meets the following requirements:

- **Operating System**: Windows 10+, macOS 10.14+, or Linux
- **Memory**: Minimum 4GB RAM (8GB recommended)
- **Storage**: At least 2GB of available disk space
- **Internet Connection**: Required for AI features and project synchronization

## Troubleshooting Initial Setup

### Application Won't Start
- Verify your system meets the minimum requirements
- Try running the application as an administrator (Windows) or with elevated permissions
- Check that your antivirus software isn't blocking the application

### Permission Issues
- Ensure the application has necessary file system permissions
- On macOS, you may need to allow the app in System Preferences > Security & Privacy
- On Linux, verify the executable has the correct permissions

### Network Connectivity
- Codex Editor requires internet access for AI features
- Check your firewall settings to ensure the application can connect
- Verify your network connection is stable
