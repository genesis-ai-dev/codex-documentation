## How to Export Your Translation Project

### 1. Open the Main Menu

<LoomVideo videoId="5425e3368c5d47d3844ab3658a739488" timestamp={0} title="Open the Main Menu" />

- If the main menu is not open, press the sidebar.
- Navigate to 'Project Settings'.

### 2. Locate Export Project Option

<LoomVideo videoId="5425e3368c5d47d3844ab3658a739488" timestamp={11} title="Locate Export Project Option" />

- Scroll down until you see 'Export Project'.

### 3. Choose Export Format

<LoomVideo videoId="5425e3368c5d47d3844ab3658a739488" timestamp={22} title="Choose Export Format" />

- Select the desired export format: 
  - Plain Text
  - HTML (for viewing in a browser or publishing as a static website)
  - USFM (Universal Standard Format)
