# Source and Target Languages

Every Codex project has a source language and a target language. These values affect import metadata, AI suggestions, search context, export metadata, and what collaborators expect when they open the project.

## What the Languages Mean

- **Source** is the language you are translating from.
- **Target** is the language you are translating into.

For example, if you are translating Spanish content into English, set **Source** to Spanish and **Target** to English.

## Set Languages When Creating a Project

The **Create New Project** flow asks for source and target languages before the project is created. Choose them carefully before importing files.

## Change Languages Later

1. Open the **Main Menu**.
2. Find the project details near the top.
3. Click the **Source** or **Target** language field.
4. Choose the correct language, or create a custom language if needed.
5. Sync shared projects after changing language metadata.

<Callout type="warning">
Changing language settings after translation work has started can affect AI behavior and export metadata. Tell collaborators before changing languages in a shared project.
</Callout>
