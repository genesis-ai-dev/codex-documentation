# Project Settings

Open the **Main Menu** from the list-tree icon in the activity bar. The panel is sometimes shown as **Codex Main Menu** or **Codex Project Settings** depending on context.

## Project Details

At the top of the Main Menu you can review and edit core project details:

- **Project name**: Click the editable name to rename the project.
- **Source** language: Click to change the source language.
- **Target** language: Click to change the target language.
- **Required Validations**: Set required text and audio validation counts.
- **Documents**: Open the source upload workflow.

<Callout type="warning">
Changing languages after work has started can affect AI behavior, export metadata, and collaborator expectations. Sync shared projects after changing project metadata.
</Callout>

## Collaboration and Sync

The Main Menu includes cloud and sync controls:

- **Publish to Cloud**: Create or connect the remote project so collaborators can access it.
- **Sync Settings**: Turn auto-sync on or off, choose an interval, run **Sync Now**, or set up sync for a project that is not yet connected.
- **Close Project**: Return to the startup project list.

If sync is failing, see [Sync Troubleshooting](/docs/project-management/sync-troubleshooting).

## AI and Quality Tools

- **AI Metrics** opens an analysis view for how much text has changed from AI suggestions and related project metrics.
- **Copilot Settings** opens the project system message and AI behavior settings.
- **Required Validations** controls how many text or audio approvals a cell needs.

See [AI Settings](/docs/translation/ai-settings) for details.

## Interface Settings

Use **Interface Settings** to adjust editor display behavior. Depending on the current release, this can include text size, line numbers, directionality, and whether settings apply to the current file or wider project context.

Use the current Main Menu label **Interface Settings** when asking teammates or support for help.

## Project Tools

The Tools area can include:

- **Export**: Open the Export Project wizard.
- **Import Labels**: Import cell labels for structured workflows.
- **Migration**: Open the Codex migration tool.
- **Updates**: Check for Codex app updates.
- **Release Notes**: Open the current release notes.
