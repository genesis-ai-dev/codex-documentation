## Navigating to the Main Menu

The main menu is your primary hub for navigating between different areas of Codex Editor.

To open it:
1. Look at the **top-left corner** of the interface.
2. Click the **file-tree icon** (it looks like a folder or document tree).

![Menu-icon](/images/menu-icon.png)

## Project Settings

When you open Project Settings from the main menu, the panel is organized into three main areas.

### Basic Project Information

![project-info](/images/project-info.png)

- **Title** — the name of your project, editable at any time
- **Source Language** — the language of the original text you are translating from
- **Target Language** — the language you are translating into
- **Required Validations** — configure validation requirements for your team, including separate settings for text validation and audio validation
- **Project Documents** — shows a count of the total documents in your project, with a **+** button to add more files

## Sync Settings

Below the project information, you'll find controls for keeping your project synchronized:

![sync-settings](/images/sync-settings.png)

- **Sync button** — triggers a manual sync immediately
- **Auto-sync toggle** — an on/off switch (I/O) that enables automatic background syncing
- **Sync interval** — when auto-sync is on, use the dropdown to choose how frequently syncs occur: **5 minutes**, **15 minutes**, **30 minutes**, or **1 hour**

## Project Tools

The bottom section contains a list of tools and actions for managing your project:

![project-tools](/images/project-tools.png)

- **AI Metrics** — view usage and performance data for the AI assistant
- **Import Labels** — import label sets into your project
- **Migration Tool** — tools for migrating project data (see [Migration Tool](/docs/project-management/migration-tool))
- **Copilot Settings** — configure how the AI Copilot behaves during translation (see [Configuring AI Copilot Settings](/docs/translation/ai-settings))
- **Export Project** — export your project in your chosen output format
- **Check for Updates** — check whether a newer version of Codex Editor is available
- **Text Display Settings** — opens an overlay panel on top of the current View
---
![display settings](/images/display-settings.png)

  - **File Scope** - apply display settings to Source, Target or both
  - **Update Behavior** — choose between applying settings to **all files** or **skip files with local settings**
  - **Font Size** — adjust the text size in the editor
  - **Line Numbers** — toggle line number display on or off
  - **Text Direction** — set left-to-right or right-to-left text direction
  - **Apply Changes** button at the bottom to confirm and save your display preferences
---

- **Close Project** — located at the very bottom of the list; closes the current project
