### 1. Open Command Palette

- Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac)
- This opens the command palette to access various commands

### 2. Check for Extension Updates

- Type "extensions" in the command palette
- Select "Focus on Extensions View" or "Check for Extension Updates"
- This will open the extensions panel

### 3. Locate Extensions Panel

- **Older Codex versions**: Extensions panel appears on the left sidebar
- **Newer Codex versions**: Extensions panel appears at the bottom (normally hidden)
- Look for the extensions view in either location

### 4. Find Codex Translation Editor

- Locate the "Codex Translation Editor" extension in the list
- Look for a small gear icon next to the extension name
- If an update is available, you may see a blue box indicating "Update Available"

### 5. Update the Extension

- **Option 1**: Click the update notification if visible
- **Option 2**: Click the gear icon next to the extension
- Select "Install Specific Version" to see all available versions
- Choose the latest version to install the update

### 6. Reload the Window

After updating the extension, you need to reload the window:

**Method 1 - Command Palette:**
- Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac)
- Type "Reload Window"
- Select the "Reload Window" command

**Method 2 - Menu Bar:**
- Click on the top menu bar
- Select "Show and Run Commands"
- Type "Reload Window" and execute

### 7. Verify the Update

- The window will reload with the updated extension
- Close any unnecessary panels (like the bottom extensions bar)
- Your Codex Translation Editor should now be running the latest version
- You're ready to continue working with the updated features
