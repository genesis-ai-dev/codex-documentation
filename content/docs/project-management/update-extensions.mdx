Codex has two components that can be updated independently: the **Codex application** (the editor itself) and the **Codex Translation Editor extension** (the translation features). Here's how to keep both up to date.

## Updating the Codex Application

Codex checks for application updates automatically in the background once every 24 hours. When an update is available, you'll see a notification with the option to install it.

You can also check manually:

1. Open the **Main Menu** from the sidebar
2. Scroll down to the **Tools** section and click **Updates**
3. If an update is available, follow the prompts to download and install it
4. The application will restart with the new version

## Updating the Extension

The Codex Translation Editor extension can be updated through the Extensions panel:

### Step 1: Open the Extensions Panel

Open the Extensions panel using any of these methods:

- Click the **Extensions** icon <img src="/images/extensions-icon.svg" alt="Extensions icon" style={{width: '16px', height: '16px', display: 'inline', verticalAlign: 'middle'}} /> in the sidebar
- Press `Ctrl + Shift + X` (Windows/Linux) or `Cmd + Shift + X` (Mac)
- Or open the command palette with `Ctrl + Shift + P` / `Cmd + Shift + P`, type "extensions", and select **Focus on Extensions View**

### Step 2: Find and Update the Extension

- Locate **Codex Translation Editor** in the extensions list
- Click the refresh icon at the top of the Extensions panel to check for available updates
- If an update is available, you'll see an **Update** button — click it
- Alternatively, click the gear icon next to the extension and select **Install Specific Version** to choose a particular version

### Step 3: Reload the Window

After the update installs, reload the application:

- Press `Ctrl + Shift + P` / `Cmd + Shift + P`
- Type "Reload Window" and select the command
- The window will reload with the updated extension

## Verifying Your Version

To check which version you're running:

- Open the Extensions panel (see Step 1 above)
- Find **Codex Translation Editor** in the list
- The version number is displayed next to the extension name

## Need Help?

If you encounter issues during an update:
- Try restarting the application
- Check our [Discord community](https://discord.gg/6kVJTEXYEp) for known issues with the latest release
- See the [release notes](/docs/releases/latest) for details on what changed
