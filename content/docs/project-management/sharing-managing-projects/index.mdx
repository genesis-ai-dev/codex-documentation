## Guide to Sharing and Managing Projects on Global Bible Tools

### 1. Introduction to Project Sharing

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={0} title="Introduction to Project Sharing" />

- Overview of sharing a project using a cloud-based management tool.
- Mention of the group created: Global Bible Tools.

### 2. Accessing the Group

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={47} title="Accessing the Group" />

- Navigate to the website: `git.genesisrnd.com/groups/GlobalBibleTools`.
- Note the change in domain name from FrontierRND to GenesisRND.

### 3. Managing Group Members

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={69} title="Managing Group Members" />

- Click on 'Manage Members' to add new members to the group.
- Assign roles: Developer, Maintainer, or Owner for project updates.

### 4. Creating Subgroups for Specific Languages

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={110} title="Creating Subgroups for Specific Languages" />

- Create subgroups within Global Bible Tools for specific languages (e.g., Portuguese).
- Subgroups help manage access and reduce confusion.

### 5. Publishing Projects

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={206} title="Publishing Projects" />

- Steps to create and publish a project: 
  - Initialize a new project (e.g., Test Project).
  - Set source language (e.g., Biblical Hebrew, Koine Greek) and target language (e.g., Portuguese).
  - Upload source files.

### 6. Project Settings and Publishing

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={315} title="Project Settings and Publishing" />

- Access project settings to publish the project.
- Ensure the project is published under the correct subgroup (e.g., Portuguese).

### 7. Syncing Projects

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={380} title="Syncing Projects" />

- Understand the syncing process: 
  - Projects can be synced to the cloud.
  - Manual sync option available.

### 8. Collaboration and Version Control

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={484} title="Collaboration and Version Control" />

- Multiple users can work on the same project simultaneously.
- Use version history to track changes and resolve conflicts.

### 9. Conclusion

<LoomVideo videoId="856a46aeb4f04cb7b20f273639682314" timestamp={509} title="Conclusion" />

- Recap of the process for sharing, managing, and collaborating on projects in Global Bible Tools.
