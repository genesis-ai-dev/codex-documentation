---
title: Backing Up Your Project
description: Learn how to create a backup of your project as a downloadable zip file that you can share or store safely
---

### 1. Navigate to the Project List

**If you have a project open**, open the main menu and scroll to the bottom, then click **Close Project**.

<img src="/images/main-menu.png" alt="Main menu with Close Project at the bottom" />

<img src="/images/close-project.png" alt="Close Project button at the bottom of the menu" />

**If you're on the welcome screen**, click **View Projects** to see all your available projects.

<img src="/images/view-projects.png" alt="View Projects button on the welcome screen" />

### 2. Select a Backup Format

Find the project you want to back up in the list. You'll see two backup options:

<img src="/images/zip-with-git.png" alt="ZIP with Git option" />

<img src="/images/mini-zip.png" alt="Mini ZIP option" />

|  | ZIP (with Git) | Mini ZIP |
|---|---|---|
| `includeGit` flag | `true` | `false` (omitted) |
| `.git` folder | Included | Excluded |
| File size | Larger | Smaller |
| Git history preserved | Yes | No |
| Best for | Backups or handoffs | Sharing or distribution |

Click your preferred option. The system will compress your project into a zip file that you can save anywhere and share freely.
